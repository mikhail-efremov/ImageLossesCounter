Кнопка Add new mesh set - добавить новый сет для поиска
Hits - количество попаданий искаемого цвета на изображении
Average - средний размер неприрывающихся линий попаданий
Coincidence - общее кол-во неприрывающихся линий
Lines count - кол-во линий поиска
Mesh color - цвет для отрисовки линий поиска
Searching color - искомый цвет
Sens - допуск от искоемого цвета
Sens trackbar - трэкбар для подкрутки чувствительности искомого цвета


Есть возможность работать с произвольным количеством сетов.
На данный момент попадания считаются без учёта точек, попавших в попадания других сетов.
При попытке учитывать эти попадания приложение подвисает из-за больших обьемов работ.
